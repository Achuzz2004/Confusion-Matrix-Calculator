What this confusion matrix calculator does

This confusion matrix calculator helps you generate a complete N×N confusion matrix for binary and multi-class classification problems in machine learning, deep learning, and data science workflows. By entering the number of classes, total samples, and desired accuracy, the tool automatically creates realistic diagonal (correct predictions) and off-diagonal (misclassifications) values. It also enables detailed evaluation using key classification metrics such as accuracy, precision, recall (sensitivity), specificity, and F1-score, making it ideal for model performance analysis.

Metrics

  • Precision — TP / (TP + FP)
  • Recall — TP / (TP + FN)
  • Specificity — TN / (TN + FP)
  • F1-score
  • Support

Features

  • Editable matrix cells
  • Instant recalculation
  • Python / NumPy / CSV export
  • Save as image
